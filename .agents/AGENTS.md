# Coding Standards & Guidelines for ClimateSHIELD-AI

## Architecture Principles
1. **Modular Design**: Keep the backend core platform separate from specific hazard modules.
2. **Type Safety**: Use Pydantic schemas for all API payloads and LLM outputs.
3. **Multi-Agent Coordination**: Leverage graph-based workflows to delegate tasks cleanly between specialized agents.
4. **Professional UI Aesthetics**: Stick to the dark-themed glassmorphism styles, ensuring smooth transitions and responsiveness.
